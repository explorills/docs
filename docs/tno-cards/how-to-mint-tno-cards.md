# How to Mint TNO Cards

## Desktop Browser Method

Mint directly from the preferred web browser using a Web3 wallet extension.

### Requirements

- :material-laptop: Desktop computer with Chrome, Firefox, or Brave browser
- :material-wallet: Web3 wallet extension installed (MetaMask recommended)
  - [MetaMask Extension Link](https://chromewebstore.google.com/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn){:target="_blank"}
- :material-currency-usd: Sufficient cryptocurrency balance for minting + gas fees

---

### Step-by-Step Instructions

1. **Navigate to your preferred TNO Card category:**
   - [**mint.explorills.com/#TNO-Bronze**](https://mint.explorills.com/#TNO-Bronze){:target="_blank"}
   - [**mint.explorills.com/#TNO-Silver**](https://mint.explorills.com/#TNO-Silver){:target="_blank"}
   - [**mint.explorills.com/#TNO-Gold**](https://mint.explorills.com/#TNO-Gold){:target="_blank"}
   - [**mint.explorills.com/#TNO-Diamond**](https://mint.explorills.com/#TNO-Diamond){:target="_blank"}

2. Click **"Connect Wallet"**
3. Choose network (Ethereum or Flare)
4. Select the quantity of TNO card
5. Click the **"MINT"** button
6. Confirm transaction in Web3 wallet extension
7. Wait until transaction is confirmed
8. Verify ownership by clicking **"My Address Info"**

---

2. Navigate to the browser function within your wallet app

4. Click **"Connect Wallet"**
5. Choose network (Ethereum or Flare)
6. Select the quantity of TNO card
