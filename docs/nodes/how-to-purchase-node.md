# How to Purchase a Node

## Desktop browser

Mint directly from a desktop browser using a Web3 wallet extension.

### Requirements

- :material-laptop: Desktop computer with Chrome, Firefox, or Brave
- :material-wallet: Web3 wallet extension installed (MetaMask recommended) -- [Install MetaMask](https://chromewebstore.google.com/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn){:target="_blank"}
- :material-currency-usd: Sufficient cryptocurrency balance for minting + gas fees

---

### Steps

1. Visit [node.expl.one](https://node.expl.one/#EXPL-Nodes){:target="_blank"}
2. Click **"Connect Wallet"**
3. Choose network (Ethereum or Flare)
4. Select the quantity of EXPL Nodes
5. Click **"MINT"**
6. Wait for the mint queue
7. Confirm the transaction in your wallet extension
8. Wait for the transaction to confirm on-chain
9. Click **"My Address Info"** to verify ownership
10. Click any Node ID to view it on the blockchain explorer

---

## Security best practices

!!! warning "Before you mint"
    - **Verify the URL.** Only use [node.expl.one](https://node.expl.one/){:target="_blank"} -- bookmark it.
    - **Never share your seed phrase** with anyone, for any reason.
    - **Use a hardware wallet** (Ledger, Trezor) for stronger key protection.
    - **Review transaction details** carefully before confirming.
