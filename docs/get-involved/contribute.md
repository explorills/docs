!!! quote "Open contribution model"
    The ONE ecosystem builds in the open. We accept contributions across Solidity contracts, React frontends, backend infrastructure, documentation, and community operations. If you have relevant experience in any of these areas, we want to hear from you.

---

The ecosystem spans 10+ projects sharing one token (EXPL), one identity layer (ONE id), and one revenue model. Contributors work on real products with deployed contracts and live backends.
