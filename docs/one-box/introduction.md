## Technology Stack

ONE box is built with modern web technologies for optimal performance:

- **Frontend**: React 19, TypeScript
- **Build Tool**: Bun, Vite
- **Styling**: Tailwind CSS
- **Icons**: Phosphor Icons
- **Animations**: Framer Motion
- **Deployment**: AWS S3 + CloudFront
