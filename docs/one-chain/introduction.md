!!! note "Status — Deployed"

## What ONE chain is

ONE chain is the **sovereign, gas-free EVM blockchain for the ONE ecosystem**. It is the foundational chain that the ONE projects build on:

- **Sovereign** — ONE chain runs as its own chain rather than as a layer or tenant of another network. Decisions about how it operates belong to the ONE ecosystem.
- **Gas-free** — End-user interactions are not designed to require a separate gas token in front of every action.
- **EVM** — The chain is EVM-compatible, so the broad ecosystem of Ethereum-style tools, contracts, and wallets carries over.

It is the only ONE project that has reached the **Deployed** stage at the time of writing.

## How to use it

- Visit [chain.expl.one](https://chain.expl.one) — the chain's project home.
- Use any EVM-compatible wallet to interact with addresses and contracts on the chain.
- Use ONE chain as the substrate underneath the rest of the ONE projects, in the same way the ecosystem itself is structured.

Public network parameters (chain ID, RPC endpoints, explorer URLs, and similar) belong on [chain.expl.one](https://chain.expl.one) and are not duplicated here so they cannot drift out of sync.
