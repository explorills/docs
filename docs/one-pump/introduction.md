**ONE pump** is a token sentiment tracking platform with daily voting mechanics, EXPL point rewards, and competitive leaderboards. Operates as part of the ONE ecosystem infrastructure.

**Platform**: [pump.expl.one](https://pump.expl.one){:target="_blank"}

---

## Overview

ONE pump implements a daily voting system for cryptocurrency sentiment tracking. Users vote on tokens, accumulate EXPL points through platform engagement, and participate in streak-based challenges. The platform aggregates voting data to provide community sentiment metrics.

---

## Core Features

### Daily Voting

- One pump per user, per day
- Track rankings by timeframe (daily, weekly, monthly, yearly, all-time)
- Add new cryptocurrencies to the voting pool

### EXPL Points

- **10 points** per daily pump
- **100 points** for adding a new coin
- Global leaderboard tracking
- Transparent point system

### Streak Challenges

- Track consecutive days pumping
- **Streak Freeze**: Protect your streak for 50 EXPL
- **Streak Challenges**: Bet points on maintaining goals
- Streak-based leaderboards

### Your Profile

- View complete pump history
- Track coins you've added to the platform
- Monitor your rank and point balance
- Analyze your engagement patterns

---

## How It Works

1. **Connect**: Link your wallet to access ONE pump
2. **Pump**: Vote for your favorite cryptocurrency once per day
3. **Earn**: Accumulate EXPL points for pumps and contributions
4. **Compete**: Climb the leaderboard and maintain your streak
5. **Discover**: Explore trending tokens based on community sentiment

---

## Technology Stack

ONE pump is built with modern web technologies for optimal performance:

- **Frontend**: React 19, TypeScript
- **Build Tool**: Bun, Vite
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Animations**: Framer Motion
- **Icons**: Phosphor Icons
- **Notifications**: Sonner

---

## Use Cases

ONE pump provides aggregated community sentiment data for cryptocurrency evaluation. The platform enables tracking of emerging tokens through voting patterns and maintains transparent, on-chain voting records. Users can analyze sentiment trends across multiple timeframes.
