## TNO Cards (Tokenized NFT Offering)

The TNO Cards program — all four tiers (Bronze, Silver, Gold, Diamond) — has been **permanently discontinued**. The `EXPL.ONE_TNO` smart contracts on both Ethereum and Flare networks have been:

- **Paused** — all minting and transfer operations halted
- **Signer key set to dead address** — public key set to `0x00dEaD`, making contract operations permanently inaccessible
- **Ownership renounced** — on both Ethereum and Flare chains

The 771,000,000 EXPL (1.38% of total supply) originally allocated to TNO Cards has been **reallocated to the Founding Contributors Royalty**, distributed gradually through EXPL Nodes. Gradual distribution replaces one-time minting, reducing sell pressure and sustaining a stable economy.
