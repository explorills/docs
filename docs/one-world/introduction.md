**ONE world** is the metaverse platform for the ONE ecosystem. Interactive 3D environments, digital identity integration, and a blockchain-backed economy — all powered by EXPL Nodes infrastructure.

**Platform**: [world.expl.one](https://world.expl.one){:target="_blank"}

---

## Overview

ONE world combines digital identity, virtual experiences, and token economics into a single platform. Built on EXPL Nodes infrastructure, it provides decentralized metaverse hosting for the ecosystem.

**Current Status**: In active development. Beta launch planned for Q4 2026.

---

## Planned Features

### Virtual Environments

- Interactive 3D spaces
- User-created content
- Social interaction zones
- Event hosting

### Digital Identity

- ONE id integration for ecosystem-wide identity
- Avatar customization
- Persistent user profiles

### Economic Integration

- EXPL token integration
- Virtual asset ownership
- Marketplace functionality
- Creator monetization

### Community Spaces

- Public gathering areas
- Private event spaces
- Guild and team coordination

---

## Core Philosophy

Like every project in the ONE ecosystem, ONE world is built on three principles:

- **Open Source** — Every line of code is publicly available on GitHub
- **Community-Driven** — Built by and for the community
- **Fair Revenue Model** — 45% to ONE Creators, 45% to EXPL Nodes, 10% to the Foundation
