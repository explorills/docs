## Allocation Breakdown

### Node-Based Distribution (77.57%)

=== "Node Operators"
    **12,500,100,000 EXPL** (22.41%)
    
    Direct rewards for Node operators. Each operator receives 171 EXPL per daily mint as their NET reward after escrow allocations.
    
    [:octicons-arrow-right-24: Learn about Nodes](../nodes/introduction.md)

=== "ONE venture"
    **12,500,100,000 EXPL** (22.41%)
    
    Strategic venture initiatives and ecosystem investments. Receives 171 EXPL per Node per daily mint via non-custodial escrow.

=== "ONE care"
    **12,500,100,000 EXPL** (22.41%)
    
    Community support and care programs. Receives 171 EXPL per Node per daily mint via non-custodial escrow.

=== "Community Grants"
    **2,500,020,000 EXPL** (4.48%)
    
    Funding for community projects and ecosystem growth initiatives. Receives 34.2 EXPL per Node per daily mint via non-custodial escrow.

=== "Developer Incentives"
    **1,250,010,000 EXPL** (2.24%)
    
    Bounties and rewards for developers building on the ONE ecosystem. Receives 17.1 EXPL per Node per daily mint via non-custodial escrow.

=== "Node Referrals"
    **1,250,010,000 EXPL** (2.24%)
    
    Incentives for Node referral program participants. Referral Nodes generate an additional 17.1 EXPL daily, sent directly to the referrer's address.

=== "Founding Contributors Royalty"
    **771,000,000 EXPL** (1.38%)
    
    Gradual royalty distribution to founding contributors through EXPL Nodes. Each of 100,000 Nodes generates +10 EXPL daily, replacing the former TNO Cards program with a sustained-release model.
    
    [:octicons-arrow-right-24: Learn about distribution](../distribution-framework/node-based-distribution.md)

---

### Direct Mint Channels (22.43%)

=== "Liquidity Mining"
    **7,500,060,000 EXPL** (13.44%)
    
    Rewards for liquidity providers in the ONE ecosystem. Activates after EXPL ERC20 deployment.

=== "Impermanent Loss Protection"
    **3,750,030,000 EXPL** (6.72%)
    
    Protection mechanism for liquidity providers against impermanent loss.

=== "HUB-Verified Users"
    **1,250,010,000 EXPL** (2.24%)
    
    Rewards for users verified through ONE id in the ONE ecosystem HUB.

=== "Mystery-Box"
    **12,500,100 EXPL** (0.02%)
    
    Community engagement prizes through Discord. Opens once every 7 days for various rewards including EXPL.
